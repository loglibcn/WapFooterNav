# WapFooterNav - Typecho移动端悬浮底部导航插件

## 功能介绍
为Typecho博客添加移动端悬浮底部导航栏，支持自定义导航项、颜色、尺寸等。

## 安装方法
1. 将插件文件夹上传到 `usr/plugins/` 目录
2. 在Typecho后台启用插件
3. 进入插件设置页面配置导航项和样式

## 配置说明
- 导航项目配置：每行一个导航项，格式为 名称|链接|图标地址
- 高亮颜色：导航项激活/悬浮时的文字颜色
- 导航栏高度：底部导航栏的高度（像素）
- 图标大小：导航图标的宽度和高度（像素）
- 日间/夜间背景色：适配不同主题模式的背景色

## 注意事项
1. 确保你的主题引入了jQuery库（大部分Typecho主题都已引入）
2. 导航图标建议使用SVG格式，适配性更好
3. 链接地址请使用相对路径或完整URL
